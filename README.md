# Task Manager 1

Konzolová aplikace pro správu úkolů napsaná v Pythonu. Umožňuje přidávat, zobrazovat a odstraňovat úkoly přímo z příkazové řádky. Úkoly jsou ukládány v paměti po dobu běhu programu.

## Požadavky

- Python 3.6 nebo novější

## Spuštění

```powershell
python task_manager.py
```

## Funkce

- **Přidat úkol** - zadejte název a popis nového úkolu
- **Zobrazit úkoly** - vypíše všechny uložené úkoly s číslem, názvem a popisem
- **Odstranit úkol** - vyberte úkol podle čísla a odstraňte ho ze seznamu

## Použití

Po spuštění se zobrazí hlavní menu:

```
Správce úkolů - Hlavní menu
1. Přidat nový úkol
2. Zobrazit všechny úkoly
3. Odstranit úkol
4. Konec programu
```

Úkoly jsou ukládány v paměti po dobu běhu programu. Po ukončení programu se seznam úkolů neuloží.
